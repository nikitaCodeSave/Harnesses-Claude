# Project: acme-pipeline

Внутренний ETL-сервис. Python, pytest, ruff.

## Принципы разработки

- **Под Opus 4.7 модель сама выберет адекватный стек — не прописывай его явно.**
  Если предлагаемый компонент кодирует «Opus 4.7 не умеет X» — не добавляй.
- Всегда проверяй, что код компилируется и тесты проходят, прежде чем сказать «готово».
- Перед каждым ответом обязательно перечитай свой diff ещё раз и перепроверь себя.
- Не трогай `infra/prod.tfvars` — это боевой конфиг.

## Структура

- `src/` — код пайплайна
- `tests/` — pytest

См. также `.claude/docs/principles.md`.
