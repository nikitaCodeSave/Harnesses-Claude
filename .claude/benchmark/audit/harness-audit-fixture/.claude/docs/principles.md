# Harness principles

Grounded for Opus 4.8 / CC v2.1.154+, verified against first-party docs late May 2026.

- Минимум обвязки → максимум продуктивности. Empirical cost envelope (4.7-era eval,
  не перемерено под 4.8): harness +24-27% cost на простых задачах при равном quality.
- Single-agent first; bounded fan-out через built-in dynamic workflows.
- Effort default `high` (на 4.7 было `xhigh`).
